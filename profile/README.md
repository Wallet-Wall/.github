# WalletWall

WalletWall is a wallet intelligence workspace for exploring holder behavior, stablecoin concentration, whale activity, and post-quantum migration readiness.

## Product surfaces

- **Whale Watcher** — wallet-level holder intelligence and whale activity analysis.
- **Stable Seer** — stablecoin holder, concentration, and leaderboard analysis.
- **Coinstellation** — token, pool, and wallet discovery graph.
- **Quantum Intelligence** — wallet-level exposure, migration readiness, and post-quantum research.

## Research

WalletWall maintains public research around post-quantum wallet migration, verifier feasibility, and vault-readiness patterns.

## Security posture

WalletWall does not custody funds, does not request seed phrases, and does not require private keys.
